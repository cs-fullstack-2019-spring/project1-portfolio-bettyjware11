  1. Javascript
    • Describes a web page's functionality/behavior
    • Lightweight interpreted or JIT-compiled programming language
    • Scripting language for Web pages
    • Prototype-basedmulti-paradigm, dynamic language, supporting object-oriented, imperative, and declarative styles.
  2. Python
    • Created by Guido van Rossum between 1985 and 1990
    • One of the easiest programming languages for novices to read and understand
    • High level general-purpose programming language
    • Supports procedural, object-oriented, and some functional programming constructs.
  3. HTML
    • HyperText Markup Language
    • Defines the meaning and structure of web content
    • "HyperText" refers to links that connect web pages to one another
    • HTML uses "markup" to annotate text, images, and other content for display in a Web browser
  4. CSS
    • Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML
    • Describes how elements should be rendered on screen, on paper, in speech, or on other media.
    • One of the core languages of the open web
    • Standardized across browsers
  5. jQuery
    • A JavaScript Library that focuses on simplifying DOM manipulation, AJAX calls, and Event handling
    • Used by JavaScript developers frequently
  6. Linux
    • Platform to run desktops, servers, and embedded systems across the globe
    • Operating system manages the communication between your software and your hardware
    • one of the most reliable, secure, and worry-free operating systems available.
  7. Ubuntu
    • An open source software operating system that runs from the desktop, to the cloud, to all your internet connected things.
    • A complete Linux operating system
